A propos de cette documentation.

But de la documentation.

Cette documentation est destinée à l'utilisateur du plugin de labelisation TraiNminaTor.Elle contient les
principales informations qui lui permettront d'utiliser au mieux les fonctionnalités proposées par le plugin.
Cette documentation est redigée au format HTML, sa visualisation est confiée à un navigateur web léger QWebView.

Vous pouvez consulter la section suivante pour les accès aux commandes usuelles de navigation de QWebView.

QWebView, navigateur web léger.

QWebView est un navigateur léger utilisé pour afficher des documents Web.
L'accès au menu contextuel des commandes de navigation (page suivante, précédente, rechargement ...), est obtenu par un clic bouton droit de la souris.
